# Leisure Centre Customer Behaviour Dataset (2024)

## Project Overview
This project presents a synthetic dataset simulating customer behaviour in a leisure centre. It includes information on visits, memberships, spending patterns, and customer demographics.

I've created this dataset to support data analysis practice and to explore real-world business scenarios such as customer retention, engagement, and revenue trends.

---

## Business Context
Leisure centres rely on understanding customer behaviour to:
- Improve member retention  
- Increase revenue  
- Optimise class schedules and facility usage  

This dataset enables analysis of these areas in a realistic but privacy-safe environment.

---

## Dataset Description
The dataset contains simulated records of:
- Customer demographics  
- Membership types  
- Visit frequency  
- Spending behaviour  
- Activity participation  

---

## Potential Use Cases
- Customer segmentation  
- Churn prediction  
- Revenue analysis  
- Peak usage trends
